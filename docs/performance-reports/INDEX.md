## How to Generate

```powershell
powershell.exe -ExecutionPolicy Bypass -File scripts/run-performance-baselines.ps1 -SkipVerifiedBuild
```

If you need a dedicated Maven repo:

```powershell
powershell.exe -ExecutionPolicy Bypass -File scripts/run-performance-baselines.ps1 -UseDedicatedRepo
```

## Report Naming

Each run also writes a timestamped report and log:

- `docs/performance-reports/performance-report-YYYYMMDD-HHMMSS.md`
- `docs/performance-reports/performance-report-YYYYMMDD-HHMMSS.log`
